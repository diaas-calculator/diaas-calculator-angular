What am I looking at?

This is a calculator that helps calculate the DIAAS value of a mix of food items. Make sure you've read the DIAAS information page first as we won't detail DIAAS basics in this documentation.

As a rule of thumb, all the proteins that come from animals contain enough of all of the indispensable amino acids but most of the proteins that come from plants don't. By choosing the right mix food items with the right quantity (to be consumed within the same meal), it is possible to balance the 9 indispensable amino acids when looking for protein sources in plants.

Ultimately, the goal of this calculator is to reduce the carbon footprint related to meat consumption. It can also be used by vegetarians/vegans who want to make sure they eat enough "complete proteins".

Language

You can switch between French and English at the top-right of the application.

For now only food item names are translated.

Contextual help

On personal computer, place the mouse over a figure to have information about what it is. Work needs to be done for mobile devices to improve the contextual help.

Listing food items, searching for food items

In the "Select food items" section you can find the food items you'll add into your mix.

When loading the page the first time, a search is automatically launched so that all the food items in our database are displayed.

You can enter a food item name and click the "Search" button to filter by name

Clicking the "Clear search" button will re-run the search with no filter.

Information about food items

In the "Select food items" section, for each food item, the below elements are displayed:

  • Name
  • DIAAS for each indispensable amino acid
  • Protein content for 100g of food

To have more information, click the "View details" button:

  • Score type. As of now possible values are:
    • DIAAS: the "regular" DIAAS, established in-vivo, on pig model, using the true ileal digestibility. It is the most precise data
    • DIAAS-rat : DIAAS on the rodents model instead of the pigs
    • DIAAS-rat-fecal : DIAAS on the rodents model and using fecal digestibility instead of true ileal digestibility
    • DIAAS-invitro : DIAAS established in vitro instead of in vivo
    • PDCAAS : PDCAAS instead of DIAAS. Included in the app when DIAAS data is not available.
  • Protein content : g of protein per 100g of food.
  • Food type
  • Cooked state for protein content : this is the cooked state related to the protein content figure provided. You should use it when you weight your ingredients.

  • Cooked state for DIAAS computation as per the study : this is the state of the product for which the DIAAS was evaluated in the study. For precise results, you should eat the product in that state. For example, if you're looking at the DIAAS of a flour: ideally the "Cooked state for protein content" should be given crude so that you can weight the flour before cooking it, and the Cooked state for DIAAS computation as per the study should be baked if you're going to bake the flour in an oven.

  • Reference : Link to the study that established the DIAAS value

  • Comment : any precision about the data provided.

Information about the mix

In the Diaas calculation for the mix of selected food items section, the first line gives global informations about the mix:

  • DIAAS value for the mix
  • Average protein content per 100g and Total protein weight (on the same column)
  • Total food weight : sum of all food weights in the mix

Then you'll find one line for each food item in the mix with:

  • Name of the food item
  • DIAAS score
  • Protein content per 100g and Protein weight in mix (on the same column)
  • Food weight in mix : quantity of food in grams present in the mix

You can use the Edit protein content button if the protein content in our database doesn't match the product you have at home. Currently, this change will not persist for future mixes.

DIAAS display

A color code is adopted to show which amino acids are missing:

  • DIAAS < 60 : red
  • 60 <= DIAAS < 75 : orange
  • 75 <= DIAAS <= 100 : yellow-green.
  • DIAAS > 100: green. Typically this is what's you'll be looking for in other food items when trying to balance a limiting amino acid.

Inside a thicker border is the value of the limiting amino acid (the lowest value of all 9 amino acids)

The BCAA names are displayed in bold. In bold and red is the leucine (the most anabolic BCAA)

For now, all DIAAS scores are given for the reference pattern of older children, adolescents and adults (> 3 years old).
